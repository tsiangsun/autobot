ALGORITHM


We use machine learning to predict price and filter out deals

For example, Toyota Camry, we show simple linear regression (left) and more advanced mixed methods including linear regression, Random Forest and K-Neighbors (right). Features considered include postdate, make, model, year, mileage, city, state, attributes (title status: clean, color, condition: excellent/good, transmission: auto/manual, fuel: gas/dissel, cylinders: v4/v6, drive: fwd/rwd/awd, ...). We choose the winner: the mixed method!
